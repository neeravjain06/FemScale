# FemScale Context
Follow this strictly. Do not deviate from API or rules.
## 5. API Specification

All endpoints prefixed /v1. Interactive docs auto-generated at GET /docs by FastAPI.
## Function Contract (MVP)

- User must provide a Python script containing:

  def handler(event):
      return result

- `event` must be JSON-serializable
- Return value must be JSON-serializable
- stdout/stderr are captured separately

#### POST /v1/jobs — Submit function for execution

Request Body (JSON)

###### 
{
  "code": "print('hello world')",
  "timeout_sec": 30,
  "input": {}
}
```
Validation Rules

- code must be a non-empty string, max 50KB — reject with 413 if exceeded
- timeout_sec: integer 1–30, default 30

Response **—** 201 Created

```
{
"job_id": "uuid4-string",
"status": "queued"
}
```
#### 5.2 GET /v1/jobs/{job_id} — Poll job status and result

Response **—** 200 OK

```
{
"job_id": "uuid4-string",
"status": "queued | running | success | failed | timeout",
"stdout": "hello world",
"stderr": "",
"error": null,
"duration_ms": 412,
"memory_mb": 34,
"cost_usd": 0.0000000023,
"created_at": "2026- 03 - 30T10:00:00Z",
"completed_at": "2026- 03 - 30T10:00:00.412Z"
}
```
Status Enum **—** Exact Strings, Used Everywhere

```
Status Meaning
queued In Redis queue, not yet picked up by a worker
```
```
running Worker has dequeued, Docker container is executing
```
```
success Execution completed, exit code 0, stdout captured
failed Execution completed, non-zero exit code, stderr captured
```
```
timeout Execution exceeded timeout_sec limit, container killed
```
Error **—** 404 Not Found

```
{ "detail": "job not found" }
```

#### 5.5 POST /v1/jobs (GitHub Actions Trigger — Stretch Goal)

No new endpoint needed. GitHub Actions calls the existing POST /v1/jobs via a curl step in the
workflow YAML:

- name: Trigger FemScale job
run: |
curl -X POST [http://localhost:8000/v1/jobs](http://localhost:8000/v1/jobs) \
- H 'Content-Type: application/json' \
- d '{"code": "print(42)", "timeout_sec": 10}'
